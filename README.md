## Getting Started

### Prerequisites

Send >= 1 OG Token to provider if this is the first time the wallet is being used.

### Setup

Create a `.env` file in the root directory:

```env
PRIVATE_KEY=<your private key without 0x prefix>
PORT=4000
NODE_ENV=development
```

### Install & Run

```bash
npm install
npm run build
npm run dev
```
